# I Chakra UI

这是一个使用 Next.js 和 Chakra UI 创建的网站。主要包含了 Chakra UI 与其他框架的集成示例。

## 开始

安装依赖：

```bash
pnpm install
```

运行开发服务器：

```bash
pnpm dev
```

用浏览器打开 [http://localhost:3000]（http://localhost:3000） 以查看结果。
